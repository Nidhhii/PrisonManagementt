About Us

  1. Home
  2. About Us

Currently, prisoner’s records are maintained in a very rudimentary way in the form of manual files and registers. This method of data management often results in human error, delay to retrieve information etc. This project provides a clear idea and details of the prison and its prisoners. The aim of this project is to reduce the amount of manual data entry and give greater efficiency thereby making tasks easier and less time consuming to its users. It helps to view and update prisoner and guard information and also helps in scheduling work for each prisoner. This project is implemented using Firebase as the database to store and manage the prisoner’s records with Python SDK for the user interface and software development. The result obtained provides a single management system which integrates all the information about a prisoner in a single profile and can easily be accessed which improves the overall efficiency of prison management.
